🚀 PASSO A PASSO PARA COLOCAR NO AR:

1️⃣ CRIAR CONTA NO RENDER
   • Acesse: https://render.com
   • Clique em "Get Started for Free"
   • Faça login com GitHub (recomendado) ou email

2️⃣ CONECTAR SEU REPOSITÓRIO GITHUB
   • No painel do Render, clique em "New +"
   • Escolha "Blueprint"
   • Clique em "Connect" no seu repositório do GitHub
   • Se não aparecer, clique em "Configure account" para dar permissão

3️⃣ DEPLOY AUTOMÁTICO
   • O Render vai detectar o arquivo render.yaml automaticamente
   • Ele vai criar 2 serviços:
     - dental-lab-db (banco de dados PostgreSQL)
     - dental-lab-app (seu site)
   • Clique em "Apply" para começar

4️⃣ AGUARDAR O DEPLOY
   • O Render vai:
     ✓ Instalar as dependências (npm install)
     ✓ Fazer o build do site (npm run build)
     ✓ Criar o banco de dados PostgreSQL
     ✓ Iniciar o servidor (npm start)
   • Isso leva uns 5-10 minutos

5️⃣ SITE NO AR! 🎉
   • Quando terminar, você terá uma URL tipo:
     https://dental-lab-app.onrender.com
   • Clique na URL para acessar seu site

═══════════════════════════════════════════════════════════════

⚠️ IMPORTANTE - LIMITAÇÕES DO PLANO GRÁTIS:

• 🛌 O site "dorme" após 15 minutos sem uso
• ⏱️ Demora ~50 segundos para "acordar" no primeiro acesso
• 🗑️ O banco de dados expira após 90 dias sem uso
• 💾 Limite de 1GB de armazenamento no banco
